Marketing Agency

In this project, I was issued with a starter code and had to rectify existing code to meet certain standards, in an attempt of making it more accessible for the client. Web accessibility is key in the digital world, as you are ensuring all potential users are able to access information by eliminating any existing barriers.

A few examples of the modifications made to the webpage included:

  • Ensuring all heading elements where in sequential order
  • Providing alt attributes to all images for accessibility
  • Using anchor tags to links to sections of the page

    • Overall, the project has enhanced my understanding of CSS and HTML and how the two when used collectively can provide functioning yet accessible webpages.
